# Spotify Clone

## Overview
This project aims to replicate the popular music streaming service, Spotify, providing users with a similar interface and experience.

## Features
- User Authentication
- Playlist Management
- Music Search
- Responsive Design

## Technologies Used
- React
- Node.js
- Express
- MongoDB

## Getting Started
To run the project locally, follow these steps:
1. Clone the repository
2. Install dependencies
3. Start the server
